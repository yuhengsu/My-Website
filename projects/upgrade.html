Up-Grade

Up-Grade is an education application designed to help over 1000 students improve their academic skills. The application allows the user to keep track of their course grade and assignment grade while calculating the user's overall GPA. The application also have additional features such as a calculator, a calender, as well as a note taking system for the user to leave notes for themselves in their classes
